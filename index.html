Welcome to My Maths Practice Page

This page contains practice sets for class 10 students.



Practice Set – Identify a and b

✅ Level 1 – Very Simple

  1. 3x + 5
  2. 7x − 2
  3. −4x + 9
  4. x + 6
  5. −x − 3

✅ Level 2 – With Rearranging Needed

  1. 5 + 2x
  2. 8 − 3x
  3. −6 + x
  4. 12 − x
  5. −9 − 4x

✅ Level 3 – Brackets (Single Term Outside)

  1. 2(x + 3)
  2. 3(x − 5)
  3. −4(x + 2)
  4. 5(2x + 1)
  5. −2(3x − 4)

✅ Level 4 – Two Brackets / Combining Like Terms

  1. 2(x + 3) + 4
  2. 3(x − 2) − 5
  3. 4x + 3 − 2x + 7
  4. 5(x + 1) − 2x
  5. 2(3x − 1) + 4x

✅ Level 5 – Slightly Challenging

  1. 3(2x − 5) − 4(x + 1)
  2. 5(x − 2) + 3(2x + 1)
  3. −2(3x − 4) + 5
  4. 4(x + 3) − 2(2x − 1)
  5. 6 − 3(2x + 1)

Practice Set – Create Your Own Expression (Form ax + b)

✅ Level 1 – Direct Creation

  1. Create an expression where

    • a = 4
    • b = 7

  2. Create an expression where

    • a = −3
    • b = 5

  3. Create an expression where

    • a = 1
    • b = −6

  4. Create an expression where

    • a = −1
    • b = −8

  5. Create an expression where

    • a = 10
    • b = 0

✅ Level 2 – Word Clues (Translate to Expression)

  1. A number multiplied by 5 and then increased by 3

  2. A number decreased by 4 and then multiplied by 2

  3. Three times a number minus 7

  4. Eight more than twice a number

  5. Six less than four times a number

✅ Level 3 – Reverse Thinking

  1. Write an expression where

    • coefficient of x is negative
    • constant term is positive

  2. Write an expression where

    • both a and b are negative

  3. Write an expression where

    • a is even
    • b is odd

  4. Write an expression where

    • a is a fraction
    • b is an integer

  5. Write an expression where

    • after simplifying, a = 5 and b = −3 (Hint: use brackets)

✅ Level 4 – Real-Life Framing

  1. Taxi charges ₹20 fixed plus ₹15 per km. Write the expression.

  2. A shopkeeper gives ₹5 discount on each item priced x. Write the expression.

  3. A salary of ₹2000 plus ₹50 per hour work. Write the expression.

  4. A mobile recharge plan costs ₹99 plus ₹2 per minute. Write the expression.

  5. A water tank already has 100 liters and fills at 8 liters per minute. Write the expression.
